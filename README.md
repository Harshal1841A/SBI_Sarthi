# Sarthi AI - Agentic Banking Platform for SBI

Sarthi is an advanced multi-agent banking AI platform built for State Bank of India (SBI), featuring voice assistants, vernacular language support, Human-in-the-Loop (HITL) supervisor workflows, and prompt shield protection.

## Live Demo Mode
This space is configured with **Demo Mode Active**. Visitors can explore the onboarding workflows and inspect the HITL Supervisor queue in real-time.

## Architecture
This Hugging Face Space hosts both the React Vite frontend UI and FastAPI Python backend in a unified Docker container running on port `7860`.
